== Description ==

SHORDIO – WORDPRESS PLUGIN DESCRIPTION
The Shordio Audio Player plugin is an HTML5+ Flash Hybrid-based WordPress plugin, which can be used to embed a recording from Shordio using a shortcode generated once the shordy is created through the Shordio app or website.   The audio player will play on all major browsers. 

FEATURES
1.  The audio player is compact, so it takes up only a small space on your webpage.
2.  It is HTML5 compatible, so the audio files embedded with this plugin will play on iOS devices.
3.  It works on all major browsers.
4.  The audio player is responsive.
5.  The audio player can be added to any post or page using shortcode generated on the Shordio app or website.

PLAYER FUNCTIONALITIES
1.  Play shordy
2.  Pause player
3.  15 seconds of forward and backward
4.  Play speed adjustments to 0.5x, 1x, 1.5x

FUNCTIONALITIES AVAILABLE AFTER LOGIN
1.  Subscribe/Unsubscribe to the writer/publisher of the shordy
2.  Add to/Remove from Favorites
3.  Add to/Remove from Playlist

USAGE
Use the shortcode generated on Shordio whenever a shordy is created, found by tapping or clicking the embed (</>) symbol.  

INSTALLATION
1.  Activate the plugin through the “Plugins” menu in WordPress.
2.  Create a shordy on Shordio, then find the WordPress shortcode under the embed link (</>).
3.  Add the shortcode anywhere on the WordPress post or page.


== Frequently Asked Questions ==

= What does this plugin do ? = 

This plugin provides the ability to embed an mp3 audio file on your WordPress post or page using a shortcode.

= How to add Shortcode? =

Answer: You can add shortcode from version 1.0 onwards

= In PHP =

`<?php echo do_shortcode('[shordio_player code="MPNtWVIek"]'); ?>

with ad

`<?php echo do_shortcode('[shordio_player_embed code="MPNtWVIek"]'); ?>
